# smart-house-web

Este es el frontend que consume la API de `walesp3982/smart-house-api` y está enfocado en el manejo de dispositivos IoT con funcionalidades avanzadas.

Este proyecto usa Next.js y Material UI.

**Requisitos**
- Node.js (recomendado >= 18)
- npm (incluido con Node.js)

**Clonar el repositorio**

```bash
git clone https://github.com/walesp3982/smart-house-web.git
cd smart-house-web
````

**Instalar dependencias**

```bash
npm install
```

**Comandos disponibles**

Los scripts definidos en `package.json` son:

- `dev`: inicia el servidor de desarrollo (Next.js)
- `build`: construye la aplicación para producción
- `start`: inicia la aplicación en modo producción
- `lint`: ejecuta ESLint
- `test`: ejecuta las pruebas con Vitest (modo watch)
- `test:run`: ejecuta las pruebas con Vitest (modo run)

Puedes ejecutar cualquiera con:

```bash
npm run dev
npm run build
npm start
npm run lint
npm test
npm run test:run
```

Abre http://localhost:3000 en tu navegador cuando el servidor de desarrollo esté corriendo.

**Notas rápidas**

- El proyecto usa Next.js, React, TypeScript y Material UI.
- Las pruebas usan `vitest` y `@testing-library/react`.
